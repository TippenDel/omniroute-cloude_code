# Claude Code через OmniRoute

Пошаговая инструкция по установке и настройке Claude Code с использованием OmniRoute и Kiro AI.

## Описание

Этот репозиторий содержит подробное руководство по:
- Установке Node.js и необходимых пакетов
- Настройке OmniRoute
- Подключению к Kiro AI
- Запуску Claude Code на macOS

## Начало работы

Полная инструкция доступна в файле [cloude_code_free.md](./cloude_code_free.md)

## Требования

- macOS
- Node.js v22.22.2 (LTS)
- Учетная запись AWS Builder ID

## Быстрый старт

```bash
# Проверка версии Node.js
node -v

# Установка OmniRoute
npm install -g omniroute

# Установка Claude Code
npm install -g @anthropic-ai/claude-code
```
